# 📝 Exam Blog (React)

Простой блог-приложение на React, реализованное как SPA с имитацией работы с API.  
Создан для демонстрации навыков маршрутизации, управления состоянием, валидации форм и обработки асинхронных операций.

## 🛠 Стек
- **React 18+** + **Vite** (сборка)
- **React Router DOM v6** (маршрутизация)
- **Zustand** (глобальное состояние)
- **React Hook Form + Zod** (формы и валидация)
- **Mock API** (имитация бэкенда с задержкой)

## 🚀 Запуск
```bash
npm install
npm run dev
